# Combat Design

## Core Philosophy

Player skill is more important than character level.

A skilled player should be able to defeat a higher-level player through better timing, positioning, and decision-making.

---

## Core Mechanics

- Sword Attack
- Dodge
- Parry
- Counter Attack
- Stamina System

---

## Combat Rules

- No auto attack
- No auto aim
- No auto dodge
- Every action depends on player input
- Good timing is rewarded

---

## Combat Flow

Enemy attacks

↓

Player chooses

- Dodge
- Parry
- Block
- Counter

↓

Successful action creates an opening.

---

## Prototype Goal

The prototype should include:

- One Player
- One Enemy
- One Sword
- HP System
- Stamina System
- Attack
- Dodge
- Parry
- Counter
